# 🔮 Ritual Bot

Um bot RPG para Discord focado em progressão, economia e sistemas avançados de interação entre jogadores.

---

## ⚔️ Principais Sistemas

### 💰 Economia
- Carteira persistente
- Sistema financeiro completo
- Mercado entre jogadores

### 📈 Progressão
- Sistema de níveis
- Experiência automática
- Recompensas exclusivas

### 🏪 Comércio
- Loja de Feiticeiros
- Loja de Maldições
- Mercado Amaldiçoado
- Sistema de Leilões

### 🎰 Cassino
- Roleta
- Jogos de aposta
- Sistema de risco e recompensa

### 📜 Pactos e Maldições
- Pactos especiais
- Benefícios exclusivos
- Consequências estratégicas

### ⚔️ Combate
- Sistema de Abates
- Rankings
- Estatísticas de jogadores

---
