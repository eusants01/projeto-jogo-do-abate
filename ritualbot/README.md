# 🩸 RitualBot

> Bot oficial do **Jogo do Abate** da Família Sant's.

## ⚔️ Funções iniciais

- Painel de entrada
- Perfil dos jogadores
- Sistema de vidas
- Registro de abates
- Ranking automático
- Banco SQLite local

## 📁 Estrutura

```txt
ritualbot/
├── main.py
├── requirements.txt
├── .env.example
├── cogs/
│   └── abate.py
└── utils/
    └── db.py
```

## 🚀 Como iniciar

1. Instale as dependências:

```bash
pip install -r requirements.txt
```

2. Crie um arquivo `.env` com base no `.env.example`.

3. Rode o bot:

```bash
python main.py
```
